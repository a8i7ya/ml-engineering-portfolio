# Machine Learning Engineering Portfolio

I am an Electronics and Computer Engineering student building a public portfolio of machine learning, applied AI, and ML engineering projects.

This repository documents my progression from ML foundations to deployed, reproducible, and production-style machine learning systems.

## Focus Areas

- Python for data science
- Data cleaning and exploratory data analysis
- Supervised machine learning
- Model evaluation and error analysis
- Deep learning
- Computer vision
- Retrieval-Augmented Generation systems
- ML deployment
- MLOps and reproducibility
- Open-source contribution
